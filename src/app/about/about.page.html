Sobre o Projeto

O Soteria é um projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Mobile. Seu objetivo é oferecer uma solução segura e intuitiva para gerenciamento de senhas em dispositivos Android.

Este aplicativo demonstra a aplicação prática de conceitos de segurança da informação, desenvolvimento mobile e experiência do usuário.
